# Калькулятор торгового эквайринга

Интерактивный веб-калькулятор для расчета стоимости услуг торгового эквайринга с выбором тарифных планов, настройкой параметров бизнеса и автоматическим расчетом ежемесячных платежей.

## Технологии

- **React 18** - библиотека для построения UI
- **TypeScript** - типизированный JavaScript
- **Tailwind CSS v4** - utility-first CSS фреймворк
- **Motion (Framer Motion)** - библиотека для анимаций
- **Vite** - сборщик и dev-сервер
- **pnpm** - пакетный менеджер

## Основные компоненты

### App.tsx
Главный компонент, содержащий всю бизнес-логику:
- Управление состоянием (обороты, терминалы, выбранный тариф)
- Расчет стоимости для каждого тарифа
- Автоматическая сортировка тарифов от выгодного к дорогому
- Drag-to-scroll карусель тарифов (ЛКМ)
- Обработка слайдеров с динамическим шагом

### BusinessSectorDropdown.tsx
Выпадающий список для выбора сферы бизнеса:
- "Для всех отраслей" (по умолчанию)
- Отраслевые тарифы (показываются только при выборе конкретной отрасли)

### TariffInfo.tsx
Отображение информации о выбранном тарифе:
- Название тарифа
- Бейджи со статусами ("ДЛЯ НОВЫХ", "ПУ ПРОСТОЙ" и др.)
- Процентные ставки
- Тултипы с дополнительной информацией

### CommissionInfo.tsx
Расчет и отображение:
- Комиссии эквайринга
- НДС (22% от комиссии)
- Итоговая сумма

### TerminalInfo.tsx
Информация о терминалах:
- Стоимость одного терминала
- Количество терминалов
- Итоговая стоимость
- Специальная логика для mPOS (единоразовый выкуп 2770₽)

## Ключевые функции

### Тарифные планы

**Тариф Стартовый** (ДЛЯ НОВЫХ)
- Для новых клиентов на первые 3 месяца
- 1% до 500 000₽, 2.2% свыше
- Бесплатная аренда терминалов
- Не поддерживает мгновенное зачисление

**Тариф Простой** (ПУ ПРОСТОЙ)
- Для клиентов с РКО ПУ Простой
- 1% + обязательная комиссия РКО +1%
- Аренда терминала 1490₽/мес (если оборот на терминал ≤ 500 000₽)
- Мгновенное зачисление: +1%

**Тариф Отраслевой**
- Для определенных отраслей
- 1.8% до 2 млн₽, 2.3% свыше
- Аренда терминала 1290₽/мес
- Не поддерживает мгновенное зачисление

**mPOS**
- Для устройства mPOS
- 1.79% фиксированная ставка
- Единоразовый выкуп терминала 2770₽
- Мгновенное зачисление: +0.20%

**Тариф Базовый**
- Для одной или нескольких торговых точек
- 2.99% без порогов
- Бесплатная аренда терминалов
- Мгновенное зачисление: +0.20%

**Тариф Доступный**
- Для небольших оборотов
- 2.5% фиксированная ставка
- Аренда терминала 890₽/мес
- Мгновенное зачисление: +0.20%

**Тариф Продвинутый**
- Для средних и крупных оборотов
- 1.99% до 2 млн₽, 2.3% свыше
- Аренда терминала 1290₽/мес
- Мгновенное зачисление: +0.20%

**Тариф Оптимальный**
- Для оборота >500 тыс₽ на терминал
- 2.2% фиксированная ставка
- Аренда терминала 1490₽/мес
- Мгновенное зачисление: +0.20%

### Слайдеры

**Оборот в месяц:**
- Диапазон: 10 000₽ - 40 000 000₽
- Минимум: 10 000₽ (автоматический возврат при blur)
- Динамический шаг: 10 000₽ до 1 млн, 100 000₽ свыше
- Ввод: свободный, с автоограничением до максимума

**Количество терминалов:**
- Диапазон: 1 - 100 шт
- Минимум: 1 шт (автоматический возврат при blur)
- Ввод: свободный, с автоограничением до максимума

### Карусель тарифов

**Drag-to-scroll:**
- Зажатие ЛКМ на карточке → свободное перемещение влево/вправо
- Отпускание → автоматическое выравнивание к ближайшей карточке
- Клик без движения (< 5px) → выбор тарифа

**Автосортировка:**
- Тарифы сортируются от самого выгодного к самому дорогому
- Мгновенная пересортировка при изменении параметров
- Плавная spring-анимация при скролле

**Навигация:**
- Кнопки влево/вправо
- Видимых карточек: 4 шт
- Размер карточки: 174×174px
- Отступ между карточками: 8px

### Тултипы

**Тултипы карточек тарифов:**
- Появляются при наведении сверху над карточкой
- Отступ от карточки: 12px
- Белый фон с тенью
- SVG-стрелочка вниз

**Тултипы бейджей:**
- "ДЛЯ НОВЫХ" → информация о сроке действия
- "ПУ ПРОСТОЙ" → требование РКО

**Тултипы полей:**
- При наведении на поля со ставками выше порога
- Объяснение условий применения ставки

### Мгновенное зачисление

Опция недоступна для тарифов:
- Тариф Стартовый
- Тариф Отраслевой

Надбавка к комиссии:
- Тариф Простой: +1%
- Остальные тарифы: +0.20%

## Особенности реализации

### Tailwind CSS v4
- Используется новая версия Tailwind v4
- Настройки темы в `/src/styles/theme.css`
- Импорты шрифтов только в `/src/styles/fonts.css`
- Не создавать `tailwind.config.js` (используется встроенная конфигурация)

### Motion (Framer Motion)
- Импорт: `import { motion } from "motion/react"`
- Используется для карусели тарифов
- Spring-анимация для плавного перехода
- Отключение анимации во время драга

### React компоненты
- Функциональные компоненты с хуками
- TypeScript для типизации props
- Controlled components для форм
- Refs для позиционирования тултипов

### Неразрывные пробелы
В тултипах используются неразрывные пробелы (`\u00A0`) для:
- Цифр с единицами измерения (3 месяца, 2770₽)
- Коротких предлогов (на, с, со, в, за)
- Предотвращения висячих предлогов

### Confetti
При нажатии кнопки "Подключить эквайринг":
- Запускается анимация конфетти
- Длительность: 3 секунды
- Выстреливает с двух сторон экрана

## Файловая структура Figma imports

Импортированные из Figma компоненты находятся в `/src/imports/`:
- **SVG-пути**: `svg-*.ts` файлы с экспортом объектов путей
- **Компоненты**: готовые React-компоненты с правильной типографикой и стилями
- **Тултипы**: специализированные компоненты для всплывающих подсказок

При работе с импортами:
- Растровые изображения: `import img from "figma:asset/hash.png"`
- SVG-файлы: `import svgPaths from "../imports/svg-name"`
- Компоненты: `import Component from "../imports/Component"`

## Разработка

### Добавление нового тарифа

1. Добавить объект в массив `tariffs` в `App.tsx`
2. Указать все параметры: rate, terminalFee, badges и т.д.
3. Добавить текст тултипа в `tariffTooltips`
4. Тариф автоматически появится в карусели

### Изменение логики расчетов

Основная функция расчета: `calculateCommission()` в `App.tsx`

Для расчета стоимости карточки: `calculateTariffPrice(tariff)`

### Стилизация

- Используйте Tailwind классы напрямую
- Для кастомных значений: inline styles
- Шрифты: `SF Pro Text`, `SF Pro Display`
